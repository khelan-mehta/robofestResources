üí° Cost Optimization Strategy & Component Justification

  • Raspberry Pi 4B: Provides sufficient computing power for ROS, SLAM algorithms, and sensor fusion while being cost-effective compared to industrial computers
  • RPLiDAR A1M8: Most expensive component but essential. Chosen over A2/A3 for significant cost savings while maintaining adequate performance for competitions
  • HC-SR04 Ultrasonics: Extremely reliable and cheap backup sensors. Three units provide 180¬∞ coverage for safety
  • Arduino Nano: Handles real-time tasks (encoder reading, motor control) that Linux on Pi cannot guarantee timing for
  • Standard Components: Using widely available parts ensures easy replacement, extensive documentation, and community support
  • Modular Design: Each subsystem can be upgraded independently as budget allows (e.g., upgrading to better LiDAR later)
  • No Overkill: Avoided expensive industrial-grade components where competition-grade alternatives suffice
